# Skill · 剧本创作（把灵感/小说打磨成可拍剧本）

> **核心信念**：观众记住的是**人和情绪**，不是画面。剧本没立住，后面分镜再美、视频再清晰，也只是几个漂亮但无关的镜头拼在一起。剧本是整部片的地基。

---

## 〇、职责边界（先认清我负责什么）
**六个阶段**（按顺序走，每个阶段做完再进下一个）：

| 阶段 | 名称 | 核心产出 |
|---|---|---|
| 一 | 人设定义 | 每个主要角色一张立体的人物小传 |
| 二 | 剧情构思 | 一条有爽点、有三幕节奏的剧情大纲 |
| 三 | 剧本生成 | 分场景的完整剧本（动作 + 逐字台词 + 钩子） |
| 四 | 多轮审查 | 结构 / 情感 / 对话 / 合规四轮过审 |
| 五 | 去 AI 味 | 把机械腔改成像人写的 |
| 六 | 交付 | 输出可被后续编导、分镜、拍摄或生成流程消费的剧本文件 |

> 创作型任务上，每个阶段做完最好**先和用户对一下方向再往下走**（人设满不满意、大纲方向对不对），避免在错误的地基上越走越远。

### ⚠️ 拿到的是一份"现成剧本"时，不等于可以跳过我

最致命的误区：用户给了一份排版工整、分集分镜头、甚至带拍摄指南的剧本，于是判定"剧本已完整"直接开拍。**"文件存在"≠"剧本过关"**——现成稿同样可能开场不抓人、三幕节奏垮、反转生硬、台词全是概述或网文 AI 腔、合规踩线。所以无论谁写的剧本：
- **一律先走阶段四多轮审查**，先给明确的"通过 / 不通过"，不通过就当场改到通过，不原样拿去拍。
- 现成稿缺人物小传或动机（只有台词没人设）就按阶段一补齐，否则下游三视图和角色一致性没依据。
- 审查 + 补齐都做完确认通过后才标记完成。**严禁"读到剧本 → 直接判完整 → 标 done → 开拍"这条捷径。**

## 一、阶段一 · 人设定义（人物是剧本的灵魂）

零散场景之所以散，往往是因为**先想画面、没先想人**。动笔写剧情前，先为每个主要角色立一张"人物小传"，让他有来历、有软肋、有想要的东西。

### 人物小传五件套（每个主要角色都要写齐）

1. **基础信息**：姓名、性别、年龄、身份/职业，以及 2-3 个**稳定可视的外貌特征**（身高体型、发型、穿着风格、标志性细节如疤痕/眼镜/配饰）。
   > 这几条后面会**直接喂给角色三视图**，越具体越不容易漂：优先写稳定可见特征（体型、服装、发型、肤色、明显痕迹、常用道具），不要只写抽象身份标签。
2. **性格特点**：核心性格 3-5 个关键词；一处**性格缺陷**（倔强 / 不善表达 / 自尊心过强）；一处**表里反差**（外表冷漠内心柔软）；一句**口头禅**或标志性语言习惯。
   > 完美的人最无聊，缺陷、反差、口头禅才让人可信、好区分。
3. **背景故事**：成长环境、影响性格的关键经历、故事开始时的处境、一个不为人知的**秘密/隐痛**。
4. **人物动机（最关键，决定他会怎么行动）**：
   - **表面目标**：他嘴上说想要的（还清债、留住对方）。
   - **深层渴望**：他真正想要的（被尊重、被看见、证明自己不是失败者）。
   - **内心恐惧**：他最怕发生的（被抛弃、被揭穿过去）。
   - **成长弧线**：他在这部片里从什么状态变成什么状态（从逃避到面对、从误解到和解）。
   > 表面目标和深层渴望/内心恐惧之间的**矛盾**，就是这个人物所有戏的发动机。哪怕几十秒的短片，主角也该有一点点变化。
5. **人物关系**：对他重要的人、与片中其他角色的关系网、敌对/对手关系，以及彼此间的旧账或张力。

### 人设创作要点

- **避免完美人设**：给缺陷和弱点，更真实。
- **制造内在冲突**：让欲望和恐惧互相打架。
- **预留成长空间**：人设要有可发展余地，不要一开始就圆满。
- **与剧情挂钩**：每个人设要素都要能在剧情里被用上、推动故事，不是写着好看。
- **差异化设计**：主要角色之间性格、说话方式、外形要能一眼分清，不要几个人像同一个。

## 二、阶段二 · 剧情构思（把一句灵感撑成有张力的主线）

拿到的常常只是一句灵感（"小情侣吵架和好""外卖员发现初恋是千金"）。把它撑成一条**有递进、有因果、有情绪起伏**的主线，而不是几个并列场景。

### 三幕节奏（短片也要有"起—升—落"的形状）

- **第一幕（小比例，别拖）**：铺设人物与处境，**开场前几秒就要抓人**——用冲突、悬念或反差把观众钩住，不要大段铺垫。
- **第二幕（占大头）**：持续升级冲突，把矛盾往人物最痛处推；每隔一小段就要有一个推动情绪的**转折/爽点**。
- **第三幕（小比例）**：给一个情绪释放点——反转、和解、爆发、顿悟；单片要有清晰落点，多段/系列则每段结尾**留一个钩子**（悬念、未解的冲突、一个意味深长的动作），让人想看下去。

> 节奏感原则：大约每隔一小段就要有一个小转折或情绪点，不要平铺直叙一条线走到底；情绪曲线要有起伏，至少安排几个能让观众共情的点（泪点 / 暖点 / 爽点）。

### 爽点库（按主题挑 8-12 个组合，用来强化主线，不是硬塞）

| 类别 | 爽点元素 |
|---|---|
| **励志奋斗** | 低谷开局、破釜沉舟、逆袭高潮、绝地反击、天赋觉醒、坚持不懈、自我突破、卧薪尝胆、逆境成长 |
| **真善美** | 善有善报、助人助己、回馈社会、正义伸张、信守承诺、知错能改、守护弱小、舍己为人、宽容大度 |
| **情感** | 爱情圆满、家人团聚、双向奔赴、破镜重圆、师徒情深、知遇之恩、跨越鸿沟、感恩回报 |
| **爽感** | 打脸羞辱、身份反转、极限翻盘、众人见证、扮猪吃虎、英雄救美、真相大白、贵人相助、化敌为友、自我救赎、甜蜜互动 |

**常见组合公式**（挑一个打底，再按故事微调）：
- **逆袭爽文**：低谷开局 + 破釜沉舟 + 绝地反击 + 逆袭高潮 + 打脸羞辱 + 身份反转 + 众人见证
- **甜宠爱情**：双向奔赴 + 英雄救美 + 误会 + 破镜重圆 + 爱情圆满 + 甜蜜互动 + 跨越鸿沟
- **热血奋斗**：低谷开局 + 天赋觉醒 + 坚持不懈 + 自我突破 + 一战成名 + 碾压对手
- **温情治愈**：善有善报 + 助人助己 + 家人团聚 + 感恩回报 + 信守承诺 + 宽容大度

> ⚠️ 爽点是**服务人物和主线**的调料，不是目的。挑能自然长在这个故事里的，别为了爽硬塞、别同一个爽点反复用，否则会变成悬浮的套路堆砌。

**本阶段产出**：一段几百字的纯叙事大纲 + 一句话标注"本片用了哪几个爽点、靠什么完成情绪释放"。

## 三、阶段三 · 剧本生成（写成可拍的剧本）

剧本最终是要喂给分镜和视频模型的，**抽象的概述喂不动模型**。把大纲拆成若干场景，每个场景都要写到"看着文字就能脑补出整段戏"的程度。

每个场景写清这几样：

- **场景**：时间、地点、环境氛围。
- **人物动作与走位**：具体到能看见——不是"两人在争吵"，而是"她攥着手机站在玄关，他低头换鞋始终没抬眼"。
- **逐字台词**：谁说、说什么、什么语气，**一字一句写出来**。绝不用"她质问了几句""两人和好了"这种概述代替——台词是后面要直接进模型的内容，概述等于没写。
- **旁白/画外音**：没有角色对白要求的剧情片、宣传片、设定展示片或概念短片，不等于没有声音叙事。除非用户明确要求纯视觉/无声，否则要判断是否需要旁白或画外音，并把需要的内容逐字写出来，让后续编导能把故事讲清。
- **结尾钩子/落点**：这一场推动了什么、留下了什么悬念或情绪。

### 台词与密度规范

- **单句别太长**，口语化、有潜台词；一段里别让一个人连续讲一大段（连续台词别超过两三行）。
- **台词占比别太高**——短片靠画面和动作叙事，不是靠对白堆，台词大致别超过全片一半。
- **凡是看不见、只能想的**（大段心理活动、抽象议论），要么转成可见的动作和表情，要么删掉。
- **每隔一小段要有一个小冲突或情绪点**；分集/分段内容要首尾呼应、每段结尾有钩子。

## 四、阶段四 · 多轮审查（写完别急着交，自己先审）

**剧本初稿写完不是终点，是审查的起点。** 和图像、视频一样，剧本也**必须由你自己当审稿人审过、改到通过，才能进入后续制作流程**——绝不允许写完一稿直接拿去拍。一个没审过的剧本喂进后面的分镜和视频流程，错误会被逐级放大、返工成本翻倍。所以这一阶段是**不可跳过的强制关卡**：每一轮带不同维度过一遍，不过关就带着问题回去改，改完再审，直到全部通过。

> **结论怎么给**：每一轮先给一句明确的**通过 / 不通过**，再逐条列问题和修改建议——不要只丢一堆分数让人猜结论。结构、情感、对话三轮明显不行就回去重写对应部分；合规任意一条踩线，无条件改。

### 第一轮 · 结构审查

- **钩子强度**：开场前几秒有没有冲突/悬念/反差，能不能拦住观众；平淡到要"耐心看下去"就是不通过。
- **节奏控制**：是不是每隔一小段就有一个转折，有没有中段拖沓注水。
- **三幕比例**：开端别太长（铺垫别超过两成）、中段对抗够厚（占大头）、结尾干脆。
- **场景连贯**：场景切换是否自然、有过渡、有逻辑，段与段是否承接；跳跃明显就要改。
- **闭环**：前面埋的伏笔/钩子，结尾有没有回收，有没有烂尾。

### 第二轮 · 情感审查

- **情感共鸣**：有没有至少几个能让观众共情的点（亲情/友情/爱情/梦想/尊严），观众能不能代入主角。
- **人物弧光**：主角从开始到结束有没有看得见的成长/内心转变，行为有没有合理动机支撑。
- **情绪曲线**：有明确起伏还是平铺直叙；压抑之后有没有释放的高潮点。
- **泪点/爽点**：有没有明确的情绪释放点，且分散在剧情里而不是堆在一处。
- **金句**：有没有一两句能被记住、能传播的台词，且落在情绪高点上。

### 第三轮 · 对话审查（短片成败的关键，单列出来重点审）⭐

台词是后面**唯一会被逐字喂进视频模型**的内容，对话不过关，前面立的人物、攒的情绪全会塌。逐句过一遍，盯这些：

- **逐字、不概述**：每句台词都是"谁说 + 说什么 + 什么语气"的完整文字，没有"她质问了几句""两人争论起来"这种偷懒的概述残留——发现一处就是不通过。
- **像人话、不像书面语**：台词口语化、有生活质感，没有"综上所述""基于以上原因"这类公文腔，也没有"不由得、心中暗道、嘴角勾起一抹"这类网文 AI 腔。
- **符合人物**：每个角色的用词、语气、口头禅要贴合他的人设和当下情绪——暴躁的人不会温吞，自卑的人不会张扬；多角色对话里，把名字遮住也能从说话方式分清谁是谁。
- **长度与密度**：单句不冗长，一个人不连续讲一大段（连续台词别超过两三行），整场台词占比别压过画面叙事（大致别超一半）。
- **有潜台词、有交锋**：对话是在"你来我往地推动冲突"，不是各说各话的信息播报；吵架戏要有情绪升级，不要平。
- **能演、能对口型**：每句都是能被演出来、能配上口型的自然语言，没有只能默念、没法说出口的内心独白混在台词里。
- **语言统一**：同一角色台词语言一致（中文就全中文），不要中英混杂（专有名词除外）。

### 第四轮 · 合规审查（一票否决，任何一条不过必须改）

- **无血腥暴力**：流血→"受伤/红肿"，厮杀→"激烈对抗"，殴打→"推搡/争执"，死亡→"离开/消失"。
- **无色情低俗**：无性暗示、暧昧擦边、粗口低俗用语。
- **无政治敏感**：不涉政治人物、事件、制度、敏感地区。
- **不抄袭**：核心剧情/人物原创，不照搬已知作品，不过度套路化。
- **价值导向健康**：传递积极正向的价值观，三观正。
- **符合平台**：符合目标投放平台的时长与内容规范。

## 五、阶段五 · 去 AI 味（让台词和叙述像人写的）

模型写的剧本常有一股机械味，台词假、叙述板。定稿前过一遍，把它改得像人写的。

| AI 味特征 | 改成 |
|---|---|
| "首先…其次…最后…" | "先说…另外…还有…" |
| "随着…的发展…" | 直接切入 |
| "综上所述…""因此…" | "说白了…""所以啊…" |
| "此外…" | "还有""对了" |
| 句子长度全都差不多 | 长短句穿插，短句出节奏、长句铺情绪 |
| 表达过于完美工整 | 加"真的""老实说""讲道理"等口语 |
| 网文腔："不由得、顿时、心中暗道、嘴角勾起一抹" | 删掉，改成看得见的动作或听得见的台词 |

**具体手法**：
- **打破完美结构**：用"然后""接着""说到这"代替"首先"，句式随机长短。
- **加生活质感**：台词允许有停顿、口头禅、没说完的话——人吵架时不会字正腔圆讲完整的道理。
- **用具体代替抽象**：与其"他很失落"，不如写一个能看见的动作或一句具体台词把失落演出来。

**收尾自检**：通篇有没有"首先/其次"？句子是不是一个长度？关键情绪有没有用具体场景演而不是说？读起来像不像机器人（大声读一遍最准）？

## 六、阶段六 · 剧本交付

剧本定稿后，输出包含 **人物小传 + 分场景剧情 + 逐字台词/旁白** 的剧本文件到工作目录。剧本文件只负责把“人、事、情绪、动作、对白/旁白、关键道具、风格方向”写扎实，不硬编码下一步必须调用哪个技能；后续由当前任务场景自行判断进入编导转译、广告拍摄、电商素材、短剧分镜、普通视频生成或其他制作流程。

> 交付时最重要的两样东西：**① 每个角色稳定具体的外貌特征**（后续视觉制作靠它锁定角色一致性）；**② 逐字台词/旁白**（后续音画生成或配音靠它保持声音叙事准确）。这两样写得够具体，后续制作才不会糊掉。

## 附 · 小说改编（用户给的是小说原文时）

如果输入是一段小说原文而非一句灵感，在上面流程基础上额外注意：

- **先判类型再改**：重生复仇（重生、复仇打脸）、霸总甜宠（误会虐恋、甜蜜互动）、悬疑惊悚（灵异、推理、恐怖氛围）等，不同类型的爽点和节奏不同，按类型强化对应套路。
- **黄金法则**：短剧改编遵循"前几秒抓眼、半分钟内见冲突、一分钟左右给反转、每段结尾留强钩子"，比普通短片更强调密度和强情绪。
- **大胆做减法**：小说的大段环境描写、心理独白、支线，**画面拍不出来的就砍**，只留能推动主线和冲突的部分；动笔前先想清楚保留什么、砍掉什么、为什么，并简要写一句"改编说明"。
- **把"叙述"翻译成"戏"**：小说里"他心里很愤怒"要改写成观众看得见的动作、表情或听得见的台词。
- **保住原作的钩子和爽点**：改编是换表达方式，不是把原文的高潮稀释掉。
