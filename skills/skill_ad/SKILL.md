# Skill · 广告创作（Brief in, Ads out）

> **何时读我**：用户的需求带有商业目的（卖货、投流、品牌曝光、种草、带货口播、出海素材）时，用我先做"广告脑"，把商业目标转成可执行的广告内容方案。
> **核心信念**：广告的第一性原理是「用内容给消费者创造意义感」。一条成功广告的本质任务 = **吸引注意力 → 激发兴趣 → 驱动商业转化**。

---

## 一、广告 ≠ 普通短片（先认清差异）

| 维度 | 普通短片(正常走) | 广告(本 skill) |
|---|---|---|
| 起点 | 故事创意 | 商业 Brief（业务目标） |
| KPI | 美感、叙事完整 | 转化 / CTR / 品牌记忆 |
| 结构 | 起承转合 | Hook → 痛点 → 产品 → 证明 → CTA |
| 主体 | 角色 | 产品 / 模特（品牌资产不可偏差） |
| 前 3 秒 | 铺垫 | 必须有视觉钩子，否则划走 |
| 产出 | 一条成片 | 常需多版本 / 多比例 / 多时长变体 |

**职责边界**：本 skill 负责 Brief → 分镜 + 广告 Prompt（广告脑）；分镜确定后的素材、生成、剪辑或投放版本制作，由当前任务场景自行选择合适流程执行。

---

## 二、Seedance 2.0 四大能力组合（按需选用）

创作前先判断这条广告要调用哪几项能力：

1. **多模态全能参考**：除文生视频、图生视频(首尾帧)，可同时上传图片/视频/音频自由组合。企业现有数字资产（设计稿、产品图、往期广告片、品牌音乐）都能作为创作起点。
   - 适用：复用老素材出新片、爆款复刻、跨境本地化一母版裂变多市场、多角度商品图一键生成展示视频。
2. **高质量音画同出**：音画同步是原生能力——口播逐字对口型、BGM 鼓点踩镜头切换、环境 ASMR 音效匹配动作，声画同源生成，省去后期配音/对口型/踩点剪辑。
   - 适用：模特图+商品图+指定语种直接生成口播带货；商品场景图自动生成匹配动作音效（倒水/开盖/撕膜）；品牌 BGM 自动踩点。
3. **意图理解与指令遵循**：按广告类型自动调度镜头切换频率、情绪推进节奏、产品出场时机；对物理规律的理解保证跨镜头人物/产品/光影一致，不变脸、不结构漂移。
   - 适用：品牌 TVC 自动按起承转合编排；种草片自动安排"痛点铺垫→产品出场"节奏。
4. **视频编辑与延长**：在已有视频上情节改写、元素增删、局部调整，保留原节奏风格精准修改；爆量素材快速裂变多投放素材。
   - 适用：爆款裂变（留结构换产品/人物）、A/B 测试（换卖点秒出多版本）、已有素材局部迭代。

## 三、广告 Prompt 公式

> **广告提示词 = 主体 + 卖点演绎 + 消费场景与调性 + 镜头语言 + 音频 + 后期约束**

普通视频公式（主体+运动+环境+运镜+美学+音频）缺了"商业意图"。广告必须补上：

**【必须】**
- **主体（人/产品）**：画面绝对视觉中心，决定受众代入感与产品认知。
- **卖点演绎（动作/剧情）**：把模糊销售话术转化为具象的微观动作或痛点场景。

**【选填】**
- **消费场景与调性**：拍摄环境/光影/美术，提供真实使用代入感。
- **镜头语言**：用专业语言控制画面，特殊视角/转场是制造"视觉钩子"和冲击力的关键。
- **音频**：音效/节奏卡点，在画面中预埋与声音贴合的视觉动作。
- **后期约束**：为后期剪辑/花字/贴纸预留视觉空间，并设反向规则（如无字幕、无水印、不变形）。

## 四、多模态 @ 语法（锁定品牌资产）

文字有想象空间，但广告里品牌视觉信息固定不容偏差。六要素均可用 `@` 语法投喂素材锁定。

**核心原则**
- **明确指代**：每引用一个素材就紧跟一句使用声明；多人物/多视频时必须明确对应关系，否则模型混淆。
  > "参考图片1中的产品外观，图片2作为场景背景，视频1参考运镜和动作节奏，音频1作为背景音乐。接下来……"
- **精准迁移**：模型自动提取参考对象核心特征，结合文本描述创作，兼顾创意与确定性/还原度。

**素材类型与控制点**
- `@图片`（锁定静态特征）：主体（产品外观/包装/Logo、模特面部/服装）、场景调性（色调/光影风格参考）、镜头（首尾帧构图/品牌落版版式）。
- `@视频`（复刻动态质感）：动作剧情（1:1 复刻微表情/动作节奏）、镜头语言（爆款运镜推拉/转场）。
- `@音频`（视听同步）：BGM 导入（按节拍调视觉动作和剪辑节奏）、人声/音效导入（口型同步/特殊声画配合）。

**积木式解耦重组示例**
> "人物面部特征参考 @视频1，发型及服装参考 @图片1，产品包装及手持动作参考 @图片2。背景环境参考 @图片3（保持高对比度光影），声音音色及语速参考 @音频1。"

**>15 秒分段生成保一致性**：用「视频延长」（上段尾帧作下段参考）或统一复用同一套 @图片/@音频 素材库。
> "根据 @视频1 的叙事节奏，延长后续内容为 xxx。"

## 五、从 Brief 到 Prompt 的三类改写思路

拿到的往往是偏业务目标的 Brief，需改写为模型可执行 Prompt。三类典型：

### 💎 品牌广告（重美学，弱卖点动作）
要点：严格锁主体，侧重消费场景调性与运镜，营造高级美学。
> 例（男士精华液 10s）：画面中央展示 @图片1 的玻璃瓶，瓶身通透比例一致【主体】；参考 @图片2 极简现代空间与高反差冷蓝调，侧光形成硬朗轮廓光【场景调性】；表面凝结水珠顺瓶身滑落展现水润保湿【卖点演绎】；微距特写开场，缓慢平稳环绕推近【运镜】；按 @音频1 鼓点节奏最后 2 秒浮现银色品牌 Logo【视听同步】；画面无跳闪、Logo 不扭曲、水珠动态真实【后期约束】。

### 📈 效果广告（视觉钩子前置，痛点外化）
要点：极强情绪开场，卖点直白，痛点外化为具体动作，不铺垫。
> 例（超薄尿不湿跑量）：参考 @视频1 快节奏运镜与焦躁神态（仅参考运镜神态，不参考原人物）；特写眉头紧锁年轻妈妈抱哭闹婴儿，极度焦虑【钩子】；第 3 秒快速横移切到 @图片1 产品【主体】；产品平铺桌面，强烈蒸汽从底部瞬间穿透表层展现透气【卖点演绎】；温馨明亮婴儿房高饱和暖调【场景调性】；动作不拖沓、蒸汽自然、包装文字清晰、五官不变形【后期约束】。

### 🔮 创意内容（特殊材质感）
要点：消费场景调性决定"特殊材质感"，卖点用微小物理交互展示产品。
> 例（猫粮毛毡定格动画）：定格动画质感、羊毛毡风格、明亮丰富、温馨童话感【场景调性】；手工毛毡橘猫（造型参考 @图片1）追红毛线球，右侧立着 @图片2 猫粮袋【主体】；扑向毛线球撞翻猫粮袋，小猫立刻低头大口吃，表现极强适口性【卖点演绎】；固定机位中景，模拟逐帧定格顿挫感【钩子】；毛毡纤维清晰、品牌名不扭曲、动作符合定格规律、不穿模【后期约束】。

## 六、高优场景最佳实践

### 📈 投流跑量（高 CTR + 快速迭代）
前 3 秒必有视觉钩子、节奏紧凑、结尾明确 CTA；用视频编辑批量换人物/换产品出变体。
- 创意前贴（毛毡/乐高/移轴动画）、口播带货（模特图+商品图+台词分秒脚本）、爆款复刻（运镜/动效复刻、口播视频）。

### 🛒 种草卖货（电商主图/详情页）
产品展示/替换为主：3C（耳机/相机）、生活用品、户外装备、服装鞋装。强调产品全程清晰不变形、多角度展示、白底或场景化。也含产地溯源（牧场奶源）、产品创意展示。

### 🌟 品牌片（情绪共鸣，不急卖货）
注重氛围营造、情绪节奏、视觉美学统一。类型：品牌概念/TVC、品牌 IP 形象延展、品牌故事表达、高奢氛围。创意意识流场景建议用 Vibe Creating 方式。

### 📱 互动带转化（KOC/UGC/短剧/热点）
真人扫脸展现真实场景。类型：达人种草、短剧剧情植入、热点借势（citywalk/节日）。强调真实记录感、口播自然、痛点反差。

### 🌏 出海素材（全球带货）
- 多语种口播：英/法/美式英语等，口播必须完整自然、语速适配时长、避免书面化过强、不需字幕。
- 多元面孔：同一产品适配不同肤色/年龄/气质，传递包容自信。

## 七、核心功能写法速查

### Slogan 与字幕生成
- **插入 Slogan**：「文字内容」+「出现时机」+「出现位置」+「出现方式」+「文字特征(颜色/风格)」。
  > 例："画面逐渐模糊，画面中部出现文字'快乐尽在 Seedance'。" / "视频开头，画面中心弹出粗体白色'买一送一'字样。"
- **加底部同步字幕**：「画面底部出现字幕，字幕内容为'…'，字幕需与音频节奏完全同步」。
- **去字幕**：「不要字幕」/「100% 透明字幕」。

### 口播带货脚本写法
按"0~Xs：动作 + 镜别 + 台词"分秒拆解；锁定人物 @图片（面部/发型/服装全程不变）、产品 @图片（外观细节不变形）、音色 @音频；明确"口型与台词一致、音色语速统一"。强调无美颜/无磨皮/眨眼频率/无饰品无麦克风等真实感约束。

### 爆款复刻 / 元素替换
明确"自动识别并跟踪每一帧目标物（含前景/中景/远景/遮挡/运动），仅替换外观（图案/颜色/品牌/造型），保持动作/人物/环境/镜头/节奏完全不变，禁止改背景/构图/光线/景深/增删物体；要求无穿帮/无漂移/无闪烁/无风格变化"。

## 八、广告方案交付

广告脑产出"分镜脚本 + 各分镜广告 Prompt + 素材清单(@引用对应关系)"后，交付给后续素材、分镜、生成、审查、剪辑或投放变体制作流程。广告片若需口播音画同出，BGM/音效要求必须作为广告方案的一部分写清楚，后续执行时逐项继承。
