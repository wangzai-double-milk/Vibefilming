# Skill · 广告创作（Brief in, Ads out）

> **何时读我**：用户的需求带有商业目的（卖货、投流、品牌曝光、种草、带货口播、出海素材）时，用我先做"广告脑"，把商业目标转成可执行的广告内容方案。
> **核心信念**：广告的第一性原理是「用内容给消费者创造意义感」。一条成功广告的本质任务 = **吸引注意力 → 激发兴趣 → 驱动商业转化**。

---

## 一、广告 ≠ 普通短片（先认清差异）

| 维度 | 普通短片(正常走) | 广告(本 skill) |
|---|---|---|
| 起点 | 故事创意 | 商业 Brief（业务目标） |
| KPI | 美感、叙事完整 | 转化 / CTR / 品牌记忆 |
| 结构 | 起承转合 | Hook → 痛点 → 产品 → 证明 → CTA |
| 主体 | 角色 | 产品 / 模特（品牌资产不可偏差） |
| 前 3 秒 | 铺垫 | 必须有视觉钩子，否则划走 |
| 产出 | 一条成片 | 常需多版本 / 多比例 / 多时长变体 |

**职责边界**：本 skill 负责 Brief → 广告策略 → 分镜与声音方案（广告脑）；分镜确定后的素材、生成、剪辑或投放版本制作，由当前任务场景自行选择合适流程执行。

---

## 二、广告方案先定四件事

创作前先判断这条广告要靠什么赢：

1. **资产复用**：是否要继承现有产品图、品牌视觉、模特形象、历史爆款节奏。
2. **音画同步**：是否需要口播、环境声、节奏卡点或品牌音乐参与转化。
3. **商业节奏**：钩子、痛点、产品出场、卖点证明、CTA 的顺序和时长分配。
4. **投放变体**：是否需要多版本、多比例、多语种、多卖点 A/B 测试。

---

## 三、广告生成要求的组成

**【必须】**
- **主体（人/产品）**：画面绝对视觉中心，决定受众代入感与产品认知。
- **卖点演绎（动作/剧情）**：把模糊销售话术转化为具象的微观动作或痛点场景。
- **商业路径**：观众为什么停下、为什么相信、为什么行动。

**【选填】**
- **消费场景与调性**：拍摄环境/光影/美术，提供真实使用代入感。
- **镜头语言**：用专业语言控制画面，特殊视角/转场是制造"视觉钩子"和冲击力的关键。
- **音频**：音效/节奏卡点，在画面中预埋与声音贴合的视觉动作。
- **后期约束**：为后期剪辑/花字/贴纸预留视觉空间，并设反向规则（如无字幕、无第三方/平台水印、不变形；这里不等于关闭参考图的「AI 生成」标识）。

---

## 四、锁定品牌资产

文字有想象空间，但广告里品牌视觉信息固定不容偏差。凡是品牌资产，必须真实作为参考素材参与生成，不能只靠文字描述。

**核心原则**
- **明确指代**：每引用一个素材就紧跟一句使用声明；多人物/多视频时必须明确对应关系，否则模型混淆。
- **精准迁移**：模型自动提取参考对象核心特征，结合文本描述创作，兼顾创意与确定性/还原度。

**素材类型与控制点**
- 图片资产：锁定产品外观、包装、Logo、模特面部、服装、场景调性、品牌落版版式。
- 视频资产：锁定动作节奏、镜头语言、表演质感、爆款结构。
- 音频资产：锁定音乐节奏、人声音色、口播语速或特殊声画配合。
- 长广告分段时，统一复用同一套品牌资产，避免每段重新发明人物、产品或音乐风格。

---

## 五、从 Brief 到可拍方案

拿到的往往是偏业务目标的 Brief，需改写为可拍方案：

- **品牌广告**：严格锁主体，侧重消费场景、视觉调性、品牌记忆点。
- **效果广告**：视觉钩子前置，痛点外化，卖点证明直白，CTA 明确。
- **创意内容**：先定材质/风格/世界观，再用微小物理交互展示卖点。
- **口播带货**：口播逐字写清，口型、语速、情绪、产品动作同步。
- **爆款复刻**：只复刻结构、节奏、运镜或表演方式，不复制不该继承的人物和品牌资产。

## 六、高优场景最佳实践

### 投流跑量
前 3 秒必有视觉钩子、节奏紧凑、结尾明确 CTA；用视频编辑批量换人物/换产品出变体。

### 种草卖货
产品展示/替换为主：3C（耳机/相机）、生活用品、户外装备、服装鞋装。强调产品全程清晰不变形、多角度展示、白底或场景化。也含产地溯源（牧场奶源）、产品创意展示。

### 品牌片
注重氛围营造、情绪节奏、视觉美学统一。类型：品牌概念/TVC、品牌 IP 形象延展、品牌故事表达、高奢氛围。创意意识流场景建议用 Vibe Creating 方式。

### 互动带转化
真人扫脸展现真实场景。类型：达人种草、短剧剧情植入、热点借势（citywalk/节日）。强调真实记录感、口播自然、痛点反差。

### 出海素材
- 多语种口播：英/法/美式英语等，口播必须完整自然、语速适配时长、避免书面化过强、不需字幕。
- 多元面孔：同一产品适配不同肤色/年龄/气质，传递包容自信。

---

## 七、核心规则

- 本版本只负责剪辑成片，不做后期字幕/叠字；广告卖点要靠口播、画面构图、产品特写本身传达。
- 口播按动作、镜别、台词拆解；锁定人物、产品、音色，保证口型与台词一致。
- 元素替换只替换指定目标，保留动作、人物、环境、镜头、节奏，不擅自改背景和构图。

---

## 八、广告方案交付

广告脑产出分镜脚本、每段生成要求、素材清单和引用关系后，交付给后续素材、分镜、生成、审查、剪辑或投放变体制作流程。广告片若需口播、BGM 或音效，必须作为广告方案的一部分写清楚，后续执行时逐项继承。
